## Es6Arrayfunctions.js
This library is made to bring the glorius es6 *arr*(.map || .filter || .reduce) into older javascript versions.
To use just link either the [Es6Arrayfunctions.js](https://oleboleskole3.github.io/HelperLibraries/Es6Arrayfunctions.js) or the minified [Es6Arrayfunctionsmin.js](https://oleboleskole3.github.io/HelperLibraries/Es6Arrayfunctionsmin.js).
